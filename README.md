Simple RTSP recorder with a user interface

Made it over the past couple days to help out some friends and as an excluse to learn about WinUI3 but might expand on it in the future depending on how things go while they use it.

WinUI3 front end with no xaml just pure C++ using this repo as an initial reference point https://github.com/sotanakamura/winui3-without-xaml/, might be a little crazy for that but the individual detail and code flow of just using C++ classes just made more sense for learning imo. Backend is a simple library written as a wrapper around ffmpeg. The RTSP streaming portion of the library was the only part of the project Claude helped a little with to speed things up, other than that everything done manually :D
